[MSG91](https://msg91.com/) is delighted to announce the launch of a new application by [Zencommerce](https://zencommerce.in/) with our SMS API integration. After this integration, we are the official SMS partner of Zencommerce. 


 


Zencommerce is a reliable platform for creating your own online store providing you with every basic feature suiting your requirements. 


 


Our **SMS API integration** services with Zencommerce provides the following benefits:


1. Enables one touch [SMS API](https://msg91.com/sms-for-developers "SMS API for developers").
2. Every client will get 500 free [transactional SMS](https://msg91.com/sms "transactional SMS").
3. 24\*7 priority support.


This post here, is a step-by-step guide, of how you can integrate our SMS APIs with Zencommerce software.


 


**Figure {1} displays how to install MSG91 add-on in** [**Zencommerce**](https://zencommerce.in/) **:**


 


* [Login](https://whmcs.zencommerce.in/clientarea.php) to your Zencommerce account.


* Click on the ‘**Application**’ option
* Go to ‘**App Store**’ [1]
* Search ‘**MSG91**’ and install the plugin
* After installation come back to “**My Application**” [2] and select the MSG91 plugin
* Now click on “**Setting**” [3] tab


 


![Image first](https://images-msg91.s3.ap-south-1.amazonaws.com/content/images/wordpress/2015/11/Image-first.png)  

   {1}


 


You will be asked for an ‘Authkey’ and you can get it on your MSG91 account 


**Figure {2} displays how to get Authkey from MSG91:**


 


* Create an account with [MSG91](https://msg91.com/signup).
* Login to this account.
* Click on “**API**” [1]  in the bottom-left corner of the panel and “**Generate**” [2] the authkey then copy this authkey
* Ask our [support superheroes](https://msg91.com/contact) for 500 free SMS


 


 


[![Image second](https://images-msg91.s3.ap-south-1.amazonaws.com/content/images/wordpress/2015/11/Image-second.png)](https://images-msg91.s3.ap-south-1.amazonaws.com/content/images/wordpress/Image-second.png)


                                                                                                 {2}


Now getting back to the Zencommerce page, 


 


**Figure {3} displays how to get the Authkey**


 


* Paste the copied authkey on ‘**Authkey**’ [1] field (e.g. 56555APUaRa5a4eada)
* Add the desired sender ID in ‘**Sender**’ [2] field ([Sender ID](http://help.msg91.com/article/40-what-is-sender-id) defines the sender (your identity) and the intent (purpose) of the message. It serves like the subject line of an e-mail.
* Add your ‘**Order Status**’ [3] for which you want to set the SMS
* Enable the tab ‘**Yes**’ [4]
* Type the SMS content related to your order status in the ‘**Message**’ [5] box, you can use TAGs for creating customizable SMS content as shown in the image below
* Save the settings
